# About #
Implementations of common solutions for [Android](http://android.com/) applications.



# Features #

## Completed features ##

  * logging (SimpleLog)
  * tasks
    * Better AsyncTasks (ProgressDialogTask)

## Developed features ##

  * WebClient
    * Support HTTP
    * JSON-RPC (2.0)

## Planned features ##

  * WebClient
    * Support TCP/IP
    * JSON-RPC, XML-RPC
  * Cache
    * FileCache
    * MemoryCache
  * Widgets
    * Better ImageView
      * Large images (without OOM)
      * Zooming
      * Scrolling
    * Common alert dialogs
    * Clickable Widgets (ImageView)
